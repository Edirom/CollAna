Detailed Instructions

Import Files

Collana allows users to import file with multiple formats such as JPEG, PNG and PDF. By clicking on this button you can see you file explorer and choose your file for sheet music comparison. Previously saved projects in form of json files can also be loaded to their saved state in this application. Multiple instances of images can be loaded one after another which will be shown in seperate image canvas along with their numbered icons on the side. Collana offers users multiple Functionalities for comparison of sheet music.

Drag & Drop

Multiple images or pdfs that are opened can be dragged and placed on top of one other. The superimposing using the drag and drop function can help to see the content on top of one another. To drag and drop function you can just click on the drag icon on a particular image canvas and move it on top of another or you can simply drag the particular image canvas numbered icon on another one.

Color Change

This function enables a user to change the forefront of the image containing characters to a more distinguishable color and the background of image to white (which acts a contrast to the forefront color for better differentiation). This function can be applied to all the opened images in different canvases seperately. The supported colors are: Red, Blue, Green, Orange, Beige. For all of these colors the background is set to white.

Zoom in/ Zoom out

All the imported source image files can be individually be zoomed in or out inside the canvas depending on the requirements for the clarity of the characters' source images. This function can be applied to individual image sources seperately by entering a zoom value in the zoom bar or using the up/down arrow in zoom bar.

Edge Detection

Edge detection is an image processing feature for finding the boundaries of any given area/object in any given source image. It basically detects and highlights all the edges in the image. This function is useful when you cannot seperate the foreground from the background. This function can be applied to individual images in different canvases seperately.

Hold

This function works when a PDF file is loaded with multiple images. This function applies the changes made on one image to all the other images in that PDF file. For instance, you make some adjustments to an image such as recoloring and the removal of the background. If you click on the 'Hold' icon, all the adjustments performed will be saved. Now you can switch to other pages in the pdf and all the previous adjustments will be applied to other images automatically.

Remove Background

This function removes the background of an image by making the background pixels transparent. The basic requirement of this feature is that the background should be initially white. If the image does not have a white background by default, you can recolor the background white by using the color change function. This function can be applied to individual images in different canvases seperately.

Perspective Transformation

It happens oftenly that some areas on the imported images are worn out/disfigured. This can be due to the conversion of the digitized material itself or it can be related to the streching of the printing plates. These individual disfigured areas in the images can be corrected with the 'Perspective Transformation' function. To use it click on this function's icon and then you can define a rectangle around the disfigured image area which you want to adjust. Now with the help of the blue dots that appear on the corners of the rectangle, the image area is arbitrarily consumed. Also, with the help of the red dot, this image area can be moved. This is useful in cases when you want to compare one clef with another clef within the image. This function can be applied to individual images in different canvases seperately.

Merge Area

This function can be used after applying the Perspective Transformation to a image. This function will allow you to merge the corrected area (which is initially shown as an overlay) with the original image. This function can be applied to individual images in different canvases seperately.

Undo

This function helps you to change the state of the image to the state before the last function applied on it. This function can be applied to individual images in different canvases seperately.

Reset

This function helps you to change the state of the image to the original state when it was initially loaded in the canvas. This function can be applied to individual images in different canvases seperately.
